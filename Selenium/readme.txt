2) Scraping with Selenium.py - The solution to the problem defined below.

- Pick a website that presents results in a paginated fashion.

- Write a script that use selenium to go over the pages on the website and download each page to a file. Store all downloaded files in a separate folder called "pages".

- The total number of pages should be determined automatically, just like we saw in the bestbuy.py example. 

- Don't pick BestBuy or Amazon

- Don't pick a website with less than 2 or over 5 pages.
